## Introduction

This project is a webpage that takes the user queries and process the response from ChatGPT. Here we use Flask framework, Python for backend and React for frontend.

## Features

**Chatbot Backend:** Implements a Flask-based backend server to handle user queries and generate responses using the OpenAI API.
- **Chatbot Frontend:** Utilizes React.js to create a user-friendly chat interface for interacting with the chatbot.
- **Integration with OpenAI:** Incorporates the OpenAI API to leverage state-of-the-art natural language processing models for generating chatbot responses.
- **Real-time Communication:** Enables real-time communication between the frontend and backend components using RESTful APIs.
- **User Input Validation:** Implements validation checks to ensure that user input is properly formatted and processed by the chatbot.
- **Error Handling:** Implements error handling mechanisms to gracefully handle unexpected errors or exceptions during chatbot operation.
